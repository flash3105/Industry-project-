# Business Risk Portfolio Segmentation

## Capstone Project | MVA 880

This repository contains the complete R code and analysis for the Business Risk Portfolio Segmentation capstone project. The analysis discovers stable, interpretable latent risk profiles for 18,120 business clients using unsupervised learning techniques.

---

## Project Overview

**Objective:** Segment and classify business clients into distinct risk profiles without pre-existing labels, supporting credit, fraud, and relationship management decisions.

**Dataset:** 18,120 commercial client records with 37 mixed-type variables (financial health, fraud and operational risk, growth, and structural characteristics).

**Key Challenge:** High-dimensional mixed data (numeric + categorical) with missing values, coding inconsistencies, and MAR-type missingness.
